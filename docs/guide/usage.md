# 使用教程

本教程将指导您如何使用灵掌机器人的灵巧手产品完成各种操作任务。

## 基本操作

### 关节控制

1. **单关节控制**：在控制软件中选择单个关节，拖动滑块或输入角度值进行控制。

2. **多关节协同**：使用预设动作库，或通过轨迹规划实现多个关节的协同运动。

3. **力控模式**：切换到力控模式，通过设置目标力值实现精细操作。

### 抓取操作

1. **抓取规划**：使用软件中的抓取规划工具，生成最优抓取姿态。

2. **执行抓取**：
   ```python
   # 使用 Python SDK 执行抓取
   from lingzhang import RobotHand
   
   hand = RobotHand()
   hand.connect()
   hand.set_grasp_force(5.0)  # 设置抓取力为5N
   hand.grasp()
   ```

3. **释放操作**：点击软件中的「释放」按钮，或调用 SDK 中的 `release()` 方法。

## 高级功能

### 轨迹规划

通过编程方式实现复杂的轨迹规划：

```python
# 生成圆弧轨迹
waypoints = hand.generate_circle_trajectory(
    center=(0.1, 0.2, 0.3),
    radius=0.05,
    start_angle=0,
    end_angle=360,
    points=20
)

# 执行轨迹
hand.execute_trajectory(waypoints, speed=0.2)
```

### 视觉引导

结合视觉系统实现目标识别和抓取：

1. 安装并配置视觉相机
2. 运行视觉标定程序
3. 使用视觉引导功能实现自动抓取

## 故障排除

- **连接失败**：检查通信线缆是否松动，驱动是否正确安装
- **运动异常**：检查关节是否卡顿，电源供应是否稳定
- **力控失效**：检查力传感器是否校准，参数设置是否正确

如需更多帮助，请参考 [API 参考](/api/) 文档或联系技术支持。
