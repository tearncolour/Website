# LZVision 视觉系统 API 文档

欢迎使用灵掌智能 LZVision 视觉系统 API 文档。本 API 提供了与灵掌视觉系统交互的接口，用于图像处理、目标检测和视觉引导等功能。

## 概述

LZVision 是灵掌智能开发的一套完整的视觉系统解决方案，包括硬件和软件两部分。本 API 文档主要介绍软件部分的接口，用于控制和操作视觉系统。

## 功能特性

- 图像处理与分析
- 目标检测与识别
- 视觉引导与定位
- 3D 点云处理
- 实时视频流处理
- 与灵巧手控制系统无缝集成
- 支持多种相机型号

## 快速开始

### 1. 初始化视觉系统

```python
import lzvision

# 创建视觉系统实例
vision = lzvision.System()

# 初始化系统
vision.initialize()

# 加载相机配置
vision.load_camera_config("camera_config.json")
```

### 2. 获取图像

```python
# 获取实时图像
image = vision.capture_image()

# 保存图像
image.save("captured_image.jpg")
```

### 3. 图像处理

```python
# 灰度化处理
gray_image = image.convert_to_gray()

# 边缘检测
edges = gray_image.detect_edges()

# 保存处理后的图像
edges.save("edges.jpg")
```
