# 产品目录

这里是灵掌科技的产品目录，包含了我们的灵巧手产品、自动化解决方案和控制软件，为客户提供全方位的智能机器人技术支持。

## 产品系列

### 灵巧手产品

灵掌科技的灵巧手产品系列是高精度、高灵活性的机器人末端执行器，专为复杂操作任务设计，广泛应用于工业自动化、科研教育、医疗健康等领域。

#### LZ-01 灵巧手

**LZ-01** 是灵掌科技的旗舰产品，是一款具有 15 自由度的高性能机器人灵巧手，具备以下特点：

- **高度灵活**：15 个独立自由度，实现类似人类手的复杂运动
- **高精度控制**：微秒级响应时间，亚毫米级定位精度
- **强大抓取能力**：最大抓取力可达 50N，适应不同重量和形状的物体
- **智能感知**：集成多种传感器，实现力觉、触觉和视觉反馈
- **易于集成**：支持多种机器人接口，可快速集成到现有自动化系统

**应用领域**：
- 精密装配与制造
- 科研实验与教学
- 医疗辅助与康复
- 危险环境作业

[查看 LZ-01 详细规格](LZ-01/)

#### LZ-Slient 静音灵巧手

**LZ-Slient** 是一款专为对噪音敏感环境设计的低噪音灵巧手，具有以下特点：

- **超静音设计**：运行噪音低于 45dB，适合实验室和办公环境
- **轻量化结构**：重量仅 850g，减轻机器人负载
- **低功耗运行**：节能设计，延长电池续航时间
- **防水防尘**：IP65 防护等级，适应恶劣工作环境
- **模块化设计**：易于维护和升级

**应用领域**：
- 实验室自动化
- 医疗设备集成
- 消费电子测试
- 服务机器人

[查看 LZ-Slient 详细规格](LZ-Slient/)

### 自动化解决方案

灵掌科技基于灵巧手技术，提供一系列智能自动化解决方案，帮助客户实现生产过程的智能化和高效化。

#### 精密装配解决方案

针对 3C 电子、汽车零部件等行业的精密装配需求，提供高精度、高可靠性的自动化装配系统。

**主要特点**：
- 高精度定位与装配（±0.01mm）
- 自适应力控制，避免装配过程中的部件损伤
- 多品种、小批量生产的快速切换能力
- 实时质量检测与追溯
- 与 MES 系统无缝集成

#### 物料分拣解决方案

基于机器视觉和灵巧手技术，实现复杂形状和材质物料的智能分拣。

**主要特点**：
- 多相机视觉引导，实现精准识别和定位
- 自适应抓取策略，适应不同形状和表面特性的物料
- 高速分拣能力（可达 200 件/分钟）
- 可处理 1mm 至 200mm 尺寸的物料
- 模块化设计，易于扩展和定制

### 控制软件

灵掌科技提供功能强大的机器人控制软件，支持多种机器人平台和应用场景。

#### LZMotion 控制平台

**LZMotion** 是灵掌科技自主研发的机器人控制平台，提供直观的用户界面和强大的编程能力。

**主要特点**：
- 图形化编程界面，降低使用门槛
- 支持多种编程语言（Python、C++、ROS）
- 实时运动规划与控制
- 丰富的算法库（路径规划、力控制、抓取规划）
- 多机器人协同控制

[查看 LZMotion API 文档](../api/)

**主要特点**：
- 跨平台支持（Windows、Linux、macOS）
- 实时视频流传输与显示
- 远程控制与参数调整
- 数据记录与分析
- 告警与通知系统

[查看 Client 软件文档](Client/)

## 产品选型指南

选择合适的产品需要考虑以下因素：

1. **应用场景**：工业生产、科研教育、医疗健康等
2. **负载要求**：抓取物体的重量范围
3. **精度要求**：定位和操作的精度需求
4. **环境条件**：温度、湿度、噪音等环境限制
5. **集成需求**：与现有系统的兼容性
