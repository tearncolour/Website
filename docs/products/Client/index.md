# Client 客户端软件

## 产品概述

Client 是灵掌科技开发的一款轻量级机器人控制客户端软件，支持远程监控和操作机器人设备，提供直观的用户界面和丰富的功能，帮助用户轻松管理和控制机器人系统。

## 主要功能

### 1. 实时监控

- 实时显示机器人状态和运行参数
- 视频流传输与实时显示
- 传感器数据可视化
- 系统状态监控和告警

### 2. 远程控制

- 直观的拖拽式机器人控制
- 手动/自动模式切换
- 预设动作序列执行
- 多机器人协同控制

### 3. 参数配置

- 机器人参数实时调整
- 控制算法参数优化
- 传感器校准和配置
- 系统网络设置

### 4. 数据管理

- 操作日志记录和查询
- 任务数据保存和导出
- 系统配置备份和恢复
- 数据统计和分析

### 5. 告警与通知

- 实时告警系统
- 邮件/短信通知
- 告警历史记录
- 告警等级管理

## 安装与配置

### 1. 安装步骤

1. **下载安装包**
   - 从灵掌科技官网下载对应平台的安装包
   - 或通过软件中心获取

2. **运行安装程序**
   - Windows：双击安装包，按照向导完成安装
   - Linux：使用命令行安装 `sudo dpkg -i client-*.deb`
   - macOS：打开 DMG 文件，将应用程序拖入 Applications 文件夹

3. **激活软件**
   - 启动 Client 软件
   - 输入产品密钥
   - 完成在线激活

### 2. 初始配置

1. **网络配置**
   - 设置客户端网络参数
   - 配置机器人设备 IP 地址

2. **设备连接**
   - 搜索并添加机器人设备
   - 验证设备连接状态

3. **用户设置**
   - 创建用户账户
   - 设置权限级别

4. **系统参数**
   - 配置视频流参数
   - 设置告警通知方式

## 使用指南

### 1. 设备监控

- 从设备列表选择要监控的机器人
- 查看实时状态和参数
- 观察视频流和传感器数据

### 2. 机器人控制

- 切换到手动控制模式
- 使用控制面板或拖拽方式控制机器人
- 执行预设动作序列

### 3. 任务管理

- 创建新任务
- 配置任务参数
- 启动、暂停或停止任务
- 查看任务执行进度

### 4. 数据管理

- 导出操作日志
- 保存任务数据
- 分析历史数据
- 生成报告
